# Critical Flow Truth Table

Generated before implementation and updated after stabilization. Runtime/code is canonical; historical docs are secondary.

## Initial Stabilization Plan

1. Correct active FE service/page calls that pointed at dead backend routes.
2. Lock RBAC expectations for high-risk pages/actions, then align backend permissions, frontend guards, and Playwright assertions.
3. Remove Playwright harness ambiguity around auth state paths, API base URLs, proxy redirects, and seeded role assumptions.
4. Restore deterministic local gates before updating docs.
5. Document remaining contract gaps without inventing new APIs.
