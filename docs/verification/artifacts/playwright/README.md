# Playwright Test Artifacts

This directory contains Playwright test reports and artifacts.

## HTML Report

After running E2E tests, generate HTML report:

```bash
cd frontend
npx playwright test --reporter=html
npx playwright show-report
```

The HTML report will be available at the path shown by `show-report` command.

## Test Results

- `e2e-results.json` - JSON test results (if configured)
- HTML report directory - Full test report with screenshots and videos
