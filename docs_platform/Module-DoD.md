# Module Definition of Done (Template)

## Student/Public side
- [ ] Public/student routes/pages exist
- [ ] Can view own data (role-limited)
- [ ] Can submit required actions (forms/requests) with validation
- [ ] Clear status messages + error handling

## Admin/Faculty side
- [ ] Admin pages for create/update/verify/approve
- [ ] Bulk tools where needed (CSV import/export)
- [ ] Filters/search and basic dashboards
- [ ] Role-based access enforced

## IT/Server side
- [ ] Roles/permissions management for module
- [ ] Audit trail visible (who changed what, when)
- [ ] Exports/backups hooks (at least CSV export)
- [ ] Health check endpoint + basic logs surfaced

## Bridge/Core compliance
- [ ] Uses platform identity (Google SSO / local user linking)
- [ ] Uses notification service for key events
- [ ] Uses file service for uploads
- [ ] Uses API standards and versioning
